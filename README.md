# PropertySystems Frontend

This repository contains the Next.js frontend for the PropertySystems platform.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Lint the project:
   ```bash
   npm run lint
   ```

## Tech Stack

- Next.js 16.1 (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Zod
